### What I learned

- CSS Transitions, and that transition wont work for the "display" property.

```css

.card__image-overlay{
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background: var(--cyan-transparent);
    display: flex;
    flex-direction: column;
    justify-content: center;
    align-items: center;
    opacity: 0;
    transition: all .7s;
}

.card__image-container:hover .card__image-overlay{
    opacity: 1;
    cursor: pointer;
}

```
